# HIIT Timer

HIIT Timer is a simple, easily configurable timer for interval training workouts.
[Click here](http://peterthenelson.github.io/hiit) to start using it.

## Development

Pretty standard for a create-react-app single page app:

```sh
# To run the (trivial and woefully incomplete) tests:
$ npm test
# To run a local dev server
$ npm run
# To deploy to github pages
$ npm run deploy
```
